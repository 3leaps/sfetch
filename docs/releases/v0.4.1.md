## Summary
Maintenance release with security-critical dependency updates, bug fixes, and documentation improvements.

## Highlights

### Dependency Updates
- **Go runtime:** Upgraded from Go 1.23.4 to Go 1.24.0 (automatic with dependency updates)
- **Security-critical:**
  - `golang.org/x/crypto`: v0.31.0 → v0.47.0 (+16 minor versions)
  - `golang.org/x/sys`: v0.28.0 → v0.40.0 (+12 minor versions)
  - `golang.org/x/text`: v0.21.0 → v0.33.0 (+12 minor versions)
- **Other:**
  - `github.com/dlclark/regexp2`: v1.11.0 → v1.11.5

All tests pass with updated dependencies; no breaking changes detected.

### Bug Fixes
- **install-sfetch.sh:** Corrected indentation from spaces to tabs for consistency with shell formatting standards

### Documentation
- **CI/CD guide:** Clarified `GITHUB_TOKEN` usage for high-volume CI and GitHub-hosted runners to avoid 403 rate limits
- **Threat model:** Updated to reflect `SFETCH_GITHUB_TOKEN` and `GH_TOKEN` as alternative authentication environment variables
- **Makefile:** Updated `GONEAT_VERSION` to v0.5.1

## Breaking Changes
None.

## Migration
No migration required. This is a drop-in replacement for v0.4.0.

## Validation
- Full test suite passes (unit, integration, schema validation)
- Corpus tests: 39/40 passed (1 flaky HTTP test site timeout)
- Build/install/corpus cycle validated with updated dependencies

## Examples

Update sfetch to v0.4.1:
```bash
sfetch --self-update --yes
```

Or install fresh:
```bash
curl -sSfL https://github.com/3leaps/sfetch/releases/latest/download/install-sfetch.sh | bash
```
